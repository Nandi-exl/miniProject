Tema project ini adalah nabung PS5 atau bisa dibilang kredit PS5
Sumber refresni dari pembuatan adalah aplikasi bibit yang mana ada fitur menabung

Tujuan project
- Membantu audience dalam menyicil bila tidak dapat membeli PS5 Cash

Fitur yang dibutuhkan pengembangan
-Home page
pada bagian input salary belum dibuat function dimana setiap input akan mengganti value input baru
sehingga alert akan muncul satu kali saja
pada bagian button input style menggunakan position abosulte yang mana tidak lah responsive karena ukuran berdasarkan persentase 
lebar tinggi html 

-Payment page
Untuk sekarang pilihan payment 3x,6x, dan 12x digabung sebaiknya dipisahkan menjadi beberapa page html
belum dibuatkan logic jika pembayaran telah lunas dan balance menyentuh angka 0 maka input audience tidak perlu
dihitung lagi
ada rencana jika angka balance telah mneyentuh 0 maka page akan di reload secara otomatis
atau list data pembayaran akan di kosongkan kembali
